EliteRomPlus 12.0.0.3 (Occasional Build) Released for POCO F1

Features :

  • Based on Mi8's Customized vendor which is now famous for its more fluency.
  • Based on POCO's Stable ROM, Merged vendor changes from MIUI 12.0.0.3.
  • Totally Debloated, not a single useless app there.
  • New gorgeous RED Charging Animation
  • Advanced Color Reboot Menu with 6way Reboot.
  • Kept stock kernel & a little bit modified thermals for moderate usage. Use kernel's specific thermals for gaming.
  • No idle drain at all. Max 0.6% idle drain, Max 13% Active drain without heavy tasks.
  • CTS passed with/without Magisk and with/without ticking magiskhide.
  • DSB Integrated without system icons... Avoided mess below wifi/data icons.
  • F2FS filesystem support added finally... you can use F2FS filesystem now for getting a bit more speed advantage over ext4.
  • Adblocker Integrated. Use adaway app to disable it if you dont like.
  • GPU, Vulkan drivers updated to latest versions.
  • Each and every system App Updated(Except google apps)
  • Best battery backup along with performance, you wont find in any other Rom.
  • Totally Odexed system even with all the modifications. Not a single Odexed Rom exists out there other than this.
  • Disabled apk signature verification for installing vanced apks easily
  • Allow taking screenshots in any Secure App.
  • New Splash Logo for POCO specifically.
  • Disabled high volume warning.
  • Increase Systemwide volume by provided patch.
  • New Fascinating bootanimation.
  • QS Tiles arrangement can be changed on request.. currently 5x4, usually themes can change them easily...
  • More changes on the way like Activity Selector, Charging Animations, Disable any app's notifications even system apps etc.
